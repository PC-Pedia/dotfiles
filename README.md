# My .files

### Supported Softwares

* ZSH
* Tmux
* Git
* Axel

### Basic Installation


#### via curl

```shell
sh -c "$(curl -fsSL https://raw.github.com/omid/dotfiles/master/install.sh)"
```

#### via wget

```shell
sh -c "$(wget https://raw.github.com/omid/dotfiles/master/install.sh -O -)"
```
